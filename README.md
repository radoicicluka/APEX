## HorrorDB
HorrorDB is an online database of horror films. The application was created as a part of the final thesis for Bachelor's degree of Software Engineering at the University of Belgrade School of Electrical Engineering.

Three types of users can use HorrorDB with their respective authorizations. The user types are public user, registered user and administrator.

#### Public user
- view, filter and search films
- view film worker information

#### Registered user
- view, filter and search films
- view film worker information
- save films to "Watch Later"
- like/dislike films
- save liked films

#### Administrator
- view, filter and search films
- view film worker information
- save films to "Watch Later"
- like/dislike films
- save liked films
- data manipulation
  - create, edit, delete films
  - create, edit, delete tags/keywords
  - create, edit, delete collaborations
  - create, edit, delete film workers
  - create, edit, delete awards
  - create, edit, delete partners
  - create, edit, delete partnerships
  - create, edit, delete genres
  - ...

## CityBuzzer
CityBuzzer is bicycle renting service with desktop/web app for administrators and a mobile app for end users.
Only the desktop/web application was built in Oracle APEX, the mobile app was built using Swift.

The application is intended to be a portal for the administrator for data management.
It supports registration of new administrators in the application aswell as pages for managing bicycle information, reported problems and ride information.

To access the application use this [link](https://oracleapex.com/ords/r/radoicicluka/city-buzzer/login).
